# Reintroduction to Django Framework :relaxed:
This project have a crud operations with 2 tables with image

**What do you need?**
- Python 3.x
-  Django Framework:
> 	pip install Django==3.0.4

> 	python -m django --version

**For execute project**:
> 	python manage.py runserver
